# Echo Copilot

Real-time conversational AI co-pilot built for a specific family member — Phase 0 prototype.

## Tech Stack

- **Frontend**: Next.js 14 (App Router), React 18, TailwindCSS
- **State**: Zustand
- **AI**: Anthropic Claude 3.5 Sonnet with streaming
- **Icons**: Lucide React

## Features

- ✨ Real-time streaming responses
- 💬 Conversation context awareness
- 🎨 Dark-themed UI
- 📱 Responsive design
- ⚡ Fast, lightweight architecture
- 🔄 Clear conversations
- ⚙️ Settings modal

## Setup

### Prerequisites

- Node.js 18+
- Anthropic API key
